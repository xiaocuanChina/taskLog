# 新版 UI 快速开始指南

## 🎯 主要变化

### 1. 布局结构
- **左侧边栏**：显示项目统计和筛选器
- **主内容区**：显示项目列表
- **顶部工具栏**：搜索、视图切换、新建按钮

### 2. 新增功能
- ✨ **全局统计**：总项目数、进行中、已完成、总任务数
- ✨ **双视图模式**：网格视图 + 列表视图
- ✨ **状态筛选**：全部/进行中/已完成
- ✨ **多种排序**：最近更新/名称/进度/创建时间
- ✨ **进度可视化**：每个项目显示进度条
- ✨ **任务统计**：显示已完成和进行中的任务数

### 3. 视觉升级
- 🎨 深色主题（专业感更强）
- 🎨 品牌渐变色点缀
- 🎨 流畅的悬停动画
- 🎨 更清晰的信息层级

## 📖 使用指南

### 查看项目统计
左侧边栏显示：
- 总项目数
- 进行中项目数
- 已完成项目数
- 总任务数和完成进度

### 搜索项目
1. 点击顶部搜索框
2. 输入项目名称或备忘内容
3. 实时显示匹配结果

### 筛选项目
左侧边栏筛选器：
- **状态筛选**：选择"全部"、"进行中"或"已完成"
- **排序方式**：选择排序规则

### 切换视图
顶部工具栏右侧：
- 🔲 **网格视图**：适合查看详细信息
- ☰ **列表视图**：适合快速浏览大量项目

### 项目操作
点击项目卡片右上角的 ⋮ 菜单：
- ✏️ 编辑名称
- 📝 项目备忘
- 🗑️ 删除项目

### 进入项目
点击项目卡片任意位置（除了菜单按钮）即可进入项目。

## 🔧 常见问题

### Q: 如何快速找到某个项目？
A: 使用顶部搜索框，输入项目名称或备忘内容关键词。

### Q: 如何查看所有已完成的项目？
A: 在左侧边栏的状态筛选中选择"已完成"。

### Q: 网格视图和列表视图有什么区别？
A: 
- **网格视图**：显示更多详细信息，适合项目较少时使用
- **列表视图**：更紧凑，适合快速浏览大量项目

### Q: 如何按进度排序项目？
A: 在左侧边栏的排序方式中选择"按进度"。

### Q: 项目卡片上的图标是什么意思？
A: 
- 📝 表示该项目有备忘内容
- ✓ 已完成 表示所有任务都已完成

### Q: 为什么有些项目没有进度条？
A: 如果项目还没有任务，就不会显示进度条。

## 💡 使用技巧

### 1. 利用统计面板
左侧统计面板可以快速了解整体情况：
- 查看有多少项目在进行中
- 了解总体任务完成进度
- 快速定位需要关注的项目

### 2. 合理使用筛选
- 工作时筛选"进行中"项目
- 复盘时筛选"已完成"项目
- 按"最近更新"排序找到最近工作的项目

### 3. 视图切换
- 项目少时用网格视图，信息更详细
- 项目多时用列表视图，浏览更快速

### 4. 搜索优化
- 为项目添加有意义的备忘
- 使用关键词可以快速找到相关项目

### 5. 保持整洁
- 定期删除已完成的旧项目
- 使用清晰的项目命名
- 及时更新项目备忘
