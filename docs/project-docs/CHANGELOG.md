# 更新日志

## [v0.2.0] - 2026-02-04

### 🎉 重大更新：数据库存储

项目从 JSON 文件存储迁移到 SQLite 数据库存储，带来更好的性能和可靠性。

### ✨ 新增功能

- **SQLite 数据库**：使用 sql.js 实现的 SQLite 数据库存储
- **自动数据迁移**：应用启动时自动检测并迁移旧数据
- **数据库管理模块**：封装完整的数据库操作 API
- **外键约束**：确保数据完整性，支持级联删除
- **索引优化**：为常用查询添加索引，提升性能
- **测试脚本**：提供完整的数据库功能测试

### 🔧 改进

- **性能提升**：数据库查询比 JSON 文件读写更快
- **数据完整性**：外键约束和数据类型验证
- **可扩展性**：更容易添加新功能和字段
- **代码结构**：数据访问逻辑更清晰

### 📝 文档

- 新增 `DATABASE_MIGRATION.md` - 数据库迁移详细说明
- 新增 `MIGRATION_SUMMARY.md` - 迁移完成总结
- 新增 `QUICK_START.md` - 快速启动指南
- 新增 `CHANGELOG.md` - 版本更新日志
- 更新 `README.md` - 添加数据库相关说明

### 🔄 迁移说明

从 v0.1.x 升级到 v0.2.0：

1. **自动迁移**：首次启动时自动迁移数据
2. **数据备份**：旧 JSON 文件会被重命名为 `.backup` 后缀
3. **无缝切换**：无需手动操作，数据完整性得到保证

### 🛠️ 技术栈更新

- 新增依赖：`sql.js` - SQLite 数据库（纯 JavaScript 实现）

### ⚠️ 注意事项

- 建议在升级前先使用"导出数据"功能备份
- 数据库文件位于 `tasksData/tasklog.db`
- 导入导出功能保持向后兼容
