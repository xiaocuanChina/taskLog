## 🎨 样式技术

### CSS Modules
使用 CSS Modules 编写样式，避免全局污染：

```jsx
import styles from './ProjectModal.module.css'
<div className={styles.modalTitle}>...</div>
```

### CSS 变量
所有颜色使用 CSS 变量，自动适配主题：

```css
.input {
  background: var(--bg-input);
  border-color: var(--border-secondary);
  color: var(--text-primary);
}
```

### 全局样式覆盖
使用 `:global()` 覆盖 Ant Design 组件样式：

```css
:global(.ant-modal-content) {
  background: var(--bg-secondary) !important;
}
```

## 🎓 学习要点

### CSS Modules
CSS Modules 是一种 CSS 作用域隔离技术：
- 自动生成唯一的类名
- 避免全局样式污染
- 支持组合和继承
- 与 React 完美集成

### CSS 变量
CSS 自定义属性（CSS Variables）：
- 可以在运行时修改
- 支持继承和级联
- 性能优秀
- 易于维护

### 全局样式覆盖
使用 `:global()` 覆盖第三方组件样式：
- 精确控制样式优先级
- 保持样式隔离
- 易于维护
