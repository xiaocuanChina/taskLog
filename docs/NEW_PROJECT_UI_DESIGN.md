# 项目选择页面 - 全新设计方案

## 🎨 设计理念

基于 **Data-Dense Dashboard** 设计模式，打造现代化、专业的项目管理界面。

### 核心特点

1. **数据密集型布局** - 在有限空间内展示最多有用信息
2. **深色主题** - 减少视觉疲劳，突出重要内容
3. **渐变色彩** - 使用品牌渐变色（#667eea → #764ba2）
4. **流畅动画** - 所有交互都有平滑过渡效果
5. **响应式设计** - 完美适配各种屏幕尺寸

## 📐 布局结构

```
┌─────────────────────────────────────────────────────┐
│  WindowControls (顶部窗口控制栏)                      │
├──────────┬──────────────────────────────────────────┤
│          │  Toolbar (搜索、视图切换、新建按钮)        │
│          ├──────────────────────────────────────────┤
│  Sidebar │                                          │
│  (统计)  │  Main Content (项目卡片/列表)             │
│          │                                          │
│          │                                          │
└──────────┴──────────────────────────────────────────┘
```

### 左侧边栏 (280px)

- **项目概览标题**
- **统计卡片**
  - 总项目数
  - 进行中项目
  - 已完成项目
  - 总任务数 + 进度条
- **筛选器**
  - 状态筛选（全部/进行中/已完成）
  - 排序方式（最近更新/名称/进度/创建时间）

### 主内容区

#### 顶部工具栏
- 左侧：标题 + 项目数量徽章
- 右侧：搜索框 + 视图切换按钮 + 新建项目按钮

#### 内容区域
- **网格视图**：3-4列自适应网格
- **列表视图**：紧凑的列表布局

## 🎨 配色方案

### 主色调
- **背景色**：`#0a0e27` (深蓝黑)
- **卡片背景**：`rgba(255, 255, 255, 0.03)` (半透明白)
- **边框**：`rgba(255, 255, 255, 0.05)` (极淡白)
- **文字**：`#f8fafc` (亮白)

### 品牌色
- **渐变起点**：`#667eea` (紫蓝)
- **渐变终点**：`#764ba2` (紫色)

### 状态色
- **成功/已完成**：`#52c41a` (绿色)
- **警告/进行中**：`#faad14` (橙色)
- **信息/默认**：`#1890ff` (蓝色)
- **空项目**：`#d9d9d9` (灰色)

## ✨ 交互效果

### 卡片悬停
```css
transform: translateY(-4px);
box-shadow: 0 8px 24px rgba(0, 0, 0, 0.3);
border-color: rgba(255, 255, 255, 0.15);
```

### 列表项悬停
```css
transform: translateX(4px);
background: rgba(255, 255, 255, 0.05);
```

### 图标缩放
```css
.cardIcon {
  transform: scale(1.1);
}
```

### 渐变条显示
```css
.projectCard::before {
  opacity: 0 → 1;
}
```

## 📊 统计数据展示

### 全局统计
- 使用 Ant Design `Statistic` 组件
- 带图标前缀
- 彩色数值
- 总任务数下方显示进度条

### 项目统计
- 已完成任务数 (绿色勾选图标)
- 进行中任务数 (橙色时钟图标)
- 进度百分比 (渐变进度条)

## 🔍 搜索和筛选

### 搜索功能
- 实时搜索项目名称
- 搜索项目备忘内容
- 无结果时显示搜索图标空状态

### 筛选功能
- **状态筛选**：全部/进行中/已完成
- **排序方式**：
  - 最近更新（默认）
  - 按名称（中文拼音排序）
  - 按进度（从高到低）
  - 创建时间（从新到旧）

## 📱 响应式设计

### 桌面端 (>992px)
- 左侧边栏 280px
- 网格视图 3-4 列
- 完整功能展示

### 平板端 (768px - 992px)
- 左侧边栏 240px
- 网格视图 2-3 列
- 保持主要功能

### 移动端 (<768px)
- 侧边栏移至顶部，最大高度 300px
- 工具栏垂直布局
- 网格视图单列
- 列表项换行显示

## 🎯 用户体验优化

### 视觉反馈
- ✅ 所有可点击元素有 `cursor: pointer`
- ✅ 悬停状态有明显视觉变化
- ✅ 过渡动画时长 150-250ms
- ✅ 加载时淡入动画

### 信息层级
1. **最重要**：项目名称、进度百分比
2. **次要**：任务统计、日期
3. **辅助**：备忘图标、操作菜单

### 操作便捷性
- 卡片整体可点击进入项目
- 操作菜单悬停显示，减少视觉干扰
- 快捷键支持（未来可扩展）

## 🚀 性能优化

### React 优化
- 使用 `useMemo` 缓存过滤和排序结果
- 避免不必要的重渲染
- 列表使用 `key` 优化

### CSS 优化
- 使用 CSS Modules 避免样式冲突
- 使用 `transform` 而非 `top/left` 做动画
- 使用 `will-change` 提示浏览器优化

### 数据优化
- 统计数据按需计算
- 搜索防抖（可扩展）

## 📝 技术栈

- **React 18** - UI 框架
- **Ant Design 6** - 组件库
- **CSS Modules** - 样式隔离
- **Fira Sans / Fira Code** - 字体（设计系统推荐）
