# 主题切换功能使用指南

## 🎨 功能介绍

TaskLog 现在支持亮色和暗色两种主题模式，你可以根据使用环境和个人喜好自由切换。

### 支持的主题

1. **暗色主题（Dark）** - 默认主题
   - 深蓝黑背景，减少视觉疲劳
   - 适合长时间使用和夜间工作
   - 专业感强，适合开发者

2. **亮色主题（Light）** - 新增
   - 白色背景，明亮清爽
   - 适合白天和光线充足的环境
   - 传统界面风格

## 📖 使用方法

### 切换主题

1. 点击右上角的 **齿轮图标** 打开设置
2. 进入 **通用设置** 标签
3. 在 **界面主题** 部分选择你喜欢的主题
4. 主题会立即生效，无需重启应用

### 自定义主题色

除了选择主题模式，你还可以自定义主题渐变色：

1. 在设置页面找到 **主题色设置**
2. 点击 **渐变起始色** 选择器，选择你喜欢的颜色
3. 点击 **渐变结束色** 选择器，选择你喜欢的颜色
4. 实时预览效果
5. 点击 **保存** 应用更改

## 🔧 技术实现

### CSS 变量系统

主题切换基于 CSS 变量实现，所有颜色都定义在 `src/styles/themes.css` 中：

```css
/* 暗色主题 */
[data-theme="dark"] {
  --bg-primary: #0a0e27;
  --text-primary: #f8fafc;
  /* ... 更多变量 */
}

/* 亮色主题 */
[data-theme="light"] {
  --bg-primary: #f5f5f5;
  --text-primary: #000000;
  /* ... 更多变量 */
}
```

### 主题变量列表

| 变量名 | 用途 | 暗色值 | 亮色值 |
|--------|------|--------|--------|
| `--bg-primary` | 主背景色 | #0a0e27 | #f5f5f5 |
| `--bg-secondary` | 次级背景 | #0f172a | #ffffff |
| `--bg-card` | 卡片背景 | rgba(255,255,255,0.03) | #ffffff |
| `--text-primary` | 主文字色 | #f8fafc | #000000 |
| `--text-secondary` | 次级文字 | rgba(255,255,255,0.85) | rgba(0,0,0,0.88) |
| `--border-primary` | 主边框色 | rgba(255,255,255,0.05) | #e8e8e8 |
| `--color-success` | 成功色 | #52c41a | #52c41a |
| `--color-warning` | 警告色 | #faad14 | #faad14 |
| `--color-error` | 错误色 | #ff4d4f | #ff4d4f |

### 平滑过渡

主题切换时，所有颜色都会平滑过渡（0.3秒），提供流畅的视觉体验：

```css
* {
  transition-property: background-color, border-color, color;
  transition-duration: 0.3s;
  transition-timing-function: ease;
}
```

## 🐛 常见问题

### Q: 切换主题后部分组件颜色没变？
A: 刷新页面或重启应用即可。某些第三方组件可能需要重新加载。

### Q: 可以自定义更多颜色吗？
A: 当前版本仅支持自定义主题渐变色。未来版本将支持更多自定义选项。

### Q: 主题设置会丢失吗？
A: 不会。主题设置保存在配置文件中，重启应用后会自动恢复。

### Q: 可以导出/导入主题配置吗？
A: 当前版本暂不支持，未来版本将添加此功能。

### Q: 亮色主题下某些文字看不清？
A: 请检查主题色设置，避免使用过浅的颜色。建议使用推荐配色。
